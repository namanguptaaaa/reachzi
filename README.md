# ⚡ Reachzi — LinkedIn Post Intelligence

> Discover what actually reaches people on LinkedIn. Reachzi ranks posts by impression count — by sector, by profile — so you know what works before you write a single word.

Reachzi is a free, open-source tool powered by Claude AI. It works as a **standalone website** and a **Chrome extension** that auto-loads a sidebar on any LinkedIn profile you visit, showing that person's top posts ranked from highest to lowest impressions.

---

## ✨ Features

- **Impression-ranked posts** — every post sorted #1 to #6 by reach, not just likes
- **12 sectors** — AI & Tech, Marketing, Sales, Finance, SaaS, HR, Health, Creator Economy, and more
- **Profile deep-dive** — enter any creator's name to see their top posts and engagement rate
- **Viral hook analysis** — each post shows why it resonated with the audience
- **Chrome extension** — sidebar auto-loads on every LinkedIn profile you visit, zero clicks needed
- **Private by design** — your Anthropic API key is stored only in your browser, never on any server
