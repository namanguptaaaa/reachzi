# ⚡ Reachzi — LinkedIn Post Intelligence

> Discover what actually reaches people on LinkedIn. Reachzi ranks posts by impression count — by sector, by profile — so you know what works before you write a single word.

Reachzi is a free, open-source tool powered by Claude AI. It works as a **standalone website** and a **Chrome extension** that auto-loads a sidebar on any LinkedIn profile you visit, showing that person's top posts ranked from highest to lowest impressions.

---

## ✨ Features

- **Impression-ranked posts** — every post sorted #1 to #6 by reach, not just likes
- **12 sectors** — AI & Tech, Marketing, Sales, Finance, SaaS, HR, Health, Creator Economy, and more
- **Profile deep-dive** — enter any creator's name to see their top posts and engagement rate
- **Viral hook analysis** — each post shows why it resonated with the audience
- **Chrome extension** — sidebar auto-loads on every LinkedIn profile you visit, zero clicks needed
- **Private by design** — your Anthropic API key is stored only in your browser, never on any server

## 🌐 Website — Deploy to GitHub Pages (free)

1. Go to [github.com](https://github.com) and create a free account
2. Click **"New repository"** → name it `reachzi` → set to **Public** → tick "Add a README" → **Create**
3. Click **"Add file" → "Upload files"** → upload `website/index.html` renamed as `index.html`
4. Go to **Settings → Pages → Branch: main → Save**
5. Wait 60 seconds — your site is live at:

```
https://YOUR-GITHUB-USERNAME.github.io/reachzi
```

## 🧩 Chrome Extension — Setup

### Step 1 — Add icons
Create an `extension/icons/` folder and add three PNG files:
- `icon16.png` (16×16 px)
- `icon48.png` (48×48 px)
- `icon128.png` (128×128 px)

You can generate a simple orange dot logo at [favicon.io](https://favicon.io) or use any image editor.

### Step 2 — Load in Chrome
1. Open Chrome → go to `chrome://extensions`
2. Enable **Developer Mode** (toggle, top right)
3. Click **"Load unpacked"** → select the `extension/` folder
4. The Reachzi icon appears in your Chrome toolbar

### Step 3 — Add your API key
1. Click the Reachzi icon in your toolbar
2. Paste your Anthropic API key (starts with `sk-ant-`)
3. Click **"Save & activate"**

### Step 4 — Use it
Visit any LinkedIn profile at `linkedin.com/in/username` — the Reachzi sidebar loads automatically on the right side, showing posts ranked by impressions. Click the collapse button to hide it.

## 🔑 Getting an Anthropic API Key

1. Go to [console.anthropic.com](https://console.anthropic.com)
2. Sign up or log in
3. Click **"API Keys"** in the left sidebar → **"Create Key"**
4. Copy the key — it looks like `sk-ant-api03-xxxxxx...`
5. Paste it into Reachzi when prompted

New accounts get free credits. Each Reachzi lookup costs roughly $0.01 or less.

## ⚠️ Important Notes

- **AI-generated data** — since LinkedIn's public API is heavily restricted, post data is generated by Claude AI based on realistic viral patterns. Ideal for research and inspiration, not a replacement for real LinkedIn Analytics.
- **No data leaves your device** — your API key is stored in `chrome.storage.local` (extension) or `localStorage` (website).
- **Rate limits** — each profile visit triggers one Claude API call. At typical Anthropic pricing this is under $0.01 per lookup.
